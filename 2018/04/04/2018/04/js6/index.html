javascript入门基础（六）BOM

发表于 2018-04-04 | 分类于 日记   |   次阅读

摘要: JavaScript 是世界上最流行的编程语言。这门语言可用于 HTML 和 web，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

1.window对象

1>setTimeout()
超时调用(规定时间后发生事件)
setTimeout("code", millisec)或setTimeout(function(){}, millisec)
code: 代码串或者函数
millisec: 毫秒数

2>clearTimeout
清除超时调用

3>setInterval(code, millisec)
间歇调用(每隔指定时间执行一次代码)
millisec: 时间间隔的毫秒数

4>clearInterval 清除间歇调用

2.Location对象

1>location.href
返回当前页面的URL
location.href("src")

2>location.hash
返回URL中的hash

3>location.host
返回服务器名称和

4>location.hostname
返回不带端口号的服务器名称

5>location.pathname
返回URL中的目录和文件名

6>location.port
返回URL中指定的端口号

7>location.protocol
返回页面使用的协议

8>location.search
返回URL的查询字符串

9>location.replace
重新定向URL

10>location.reload
重新加载当前显示的页面

3.History对象

可用history.go(num)代替
1>history.back()
回到历史记录的上一步

2>history.forward()
回到历史记录的下一步

4.Screen对象

1>screen.availWidth
返回可用的屏幕宽度

2>screen.availHight
返回可用的屏幕高度
对比window.innerWidth与window.innerHeight

5.Navigator对象

1>navigator.userAgent
返回浏览器名称、 版本等信息
