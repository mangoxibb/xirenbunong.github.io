响应式开发（一）Foundation

发表于 2018-04-16 | 分类于 日记   |   次阅读

摘要: 号称是世界上最先进的响应式前端框架, Foundation是一个易用、强大而且灵活的框架, 用于构建基于任何设备上的Web应用. 提供多种Web上的UI 组件, 如表单、按钮、 标签等.

概览

1.开发环境的搭建

1.1 安装Git

关于git的下载地址, 安装方式, 在本人博客git入门基础中有具体讲解.

1.2 安装Node.js

查看node版本, 不能小于6.0.0(关于如何安装或者跟新node, 在本人博客"手把手教你使用Hexo打造个人博客"中有)

1.3 下载Foundation

npm install --global foundation-cli 全局安装, 所以--global

1.4 启动服务预览

cd进入到项目根目录下(非常重要！！！), 执行npm start启动服务, 然后打开文件所在目录, 在根目录下有index.html, 浏览器打开即可
