编写类的属性：属性类型 属性名称；（0或多都可）
编写类的方法

6.2构造方法

1.构造方法：是定义在java类中的一个用来初始化对象的方法，它与类同名且没有返回值。
2.语句格式：public 类名(){}
3.构造方法可分为无参和有参。
4.当没有指定构造方法时系统会自动添加无参的构造方法。
5.当有指定构造方法时，无论是有参、无参的构造方法，都不会自动添加无参的构造方法。
6.构造方法的重载：方法名相同，但参数不同的多个方法，调用时会自动根据不同的参数选择

6.3静态static

静态变量的访问可以不创建对象，
直接用 类名.静态变量 也可以创建对象用 “对象名.静态变量” 访问静态变量。
还可以通过改变 对象名.静态变量 来改变静态变量的值。
静态方法同理

6.4封装

6.4.1封装的步骤：

1、修改属性的的可见性（设置为private）   
2、创建set(接收值)和get(返回值)方法     
3、在getter和setter方法中加入属性控制语句

6.4.2 This
this 关键字，代表当前对象。
this.属性 操作当前对象的属性
this.方法 操作当前对象的方法

6.4.3内部类

当内部类和外部类属性名相同时：
使用外部类类名.this.属性名 区别 内部属性名
没有冲突时，属性名即可访问外部类属性

6.5继承

6.5.1继承的好处：

子类拥有父类的所有属性(属性前面不能是private)和方法，实现代码复用.

6.5.2语法规则

6.5.3方法的重写
为什么要方法重写?

当父类的方法体满足不了子类的需求时, 子类的方法体重写
例如人类有方法say(), 中国人类说中国话, 美国人类说美国话, 所以子类say()方法重写

在子类中重写的规则

1、返回值类型
2、方法名
3、参数类型及个数
都要与父类继承的方法相同

6.5.4final

final关键字： 表示“最终的”，即不可修改。

final可以修饰类、方法、属性和变量

1、修饰类：不允许被继承

2、修饰方法：不允许被重写

3、修饰属性：则该属性不会进行隐式初始化（不会自动初始化），需要手动初始化或者在构造方法中初始化

4、修饰变量，即变量变为常量

注意区别属性和变量(有无修饰符public等)

6.5.5super

为什么要使用super?     
子类属性和方法值不同, 但是都要使用时, super.属性 | super.方法来调用父类属性或方法
