ajax全接触

发表于 2018-05-16 | 分类于 日记   |   次阅读

摘要: AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。最大的优点是在不重新加载整个页面的情况下，可以与服务器交换数据并更新部分网页内容。

1.XMLHttpRequest发送请求有两个方法：

（1）open（method，url，async）

 method：发送请求方法；
 url：请求地址；
 async：请求同步/异步，默认值为true。

（2）send（string）

2.XML建立一个AJAX异步请求：

  1. new一个XHR对象request
  2. 调用open()方法
  3. send()一些数据
  4. 对过程进行监听来知道数据库是否响应，最后做一些事情

用Jquery实现Ajax

jQuery.ajax([settings])<br>

1:type:类型，“post”或“get”,默认为“get”；<br>

2：url:发送请求的地址<br>

3：data：是一个对象，连同请求发送到服务器的数据<br>

4：datatype：预期服务器返回的数据类型。如果不指定，
jQuery将自动根据HTTP包MIME信息来智能判断，一般我们采用json格式，
可以设置为json（可以不指定）

5：success：是一个方法，请求成功后的回调函数，传入返回后的数据，以及包含成功代码的字符串

6：error：是一个方法，请求失败时调用此函数，传入XMLHttpRequest对象
