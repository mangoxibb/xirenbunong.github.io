摘要: 层叠样式表（Cascading Style Sheet）是一种用来为结构化文档（如 HTML 文档或 XML 应用）添加样式（字体、间距和颜色等）的计算机语言. 接下来会对css中一些重要的样式进行讲解, 并分析其应用场景. Float(浮动), 在实际的网页构建中有很大的作用.

3.几个实例

3.1float: none

float默认为none

3.2float: left/right

float为left时，left脱离文档流，紫色的none盒子会上来一行

float为left和right时，因为脱离文档流所以绿色的none盒子还在同一行

3.3clear: both

关于清除浮动：

清除浮动并不是说让浮动元素回到文档流

清除浮动只会改变改元素自身的位置
并且只针对排在该元素前面的元素

绿色块独占一行：给绿色块加clear: both

紫色块也独占一行：给紫色块也加clear: both
