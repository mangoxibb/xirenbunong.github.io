2.2 Application

例如
const express = require('express');
const app = express();

方法：
app.use([path,] function [, function...])：定义一个中间件，查看路由是否以path路径开头。
中间件就是处理HTTP请求的函数，用来完成各种特定的任务，
比如检查用户是否登录、检测用户是否有权限访问等。
如果path没有写，那么默认就是"/"，同时需要注意的是：中间件的执行是有顺序的。
app.all([path,] function [, function...])：定义全局路由控制，用作路由处理，匹配完整路径，在app.use之后。
app.enable()：将参数设置为true。
app.enabled()：将参数设置为false。
app.get(path, function(req, res){})：根据请求路径来处理客户端发出的GET等各种请求。
第一个参数path为请求的路径， 第二个参数为处理请求的回调函数。
app.listen()：绑定并侦听指定主机和端口上的连接。
app.param(paramName, callback)：用来批量处理路径中的参数
app.set(key, val)：用来设置参数，例如渲染模板
app.engin()：规定文件用何种方式来进行渲染

2.3 Request

app.get('/', function (req, res) {
   res.send('Hello World');
})中的req 

req.query：获取URL的查询参数串   
req.route：获取当前匹配的路由    
req.protocol：获取协议类型   
req.path：获取请求路径

2.4 Response

app.get('/', function (req, res) {
   res.send('Hello World');
})中的res

res.send()：传送HTTP响应  
res.clearCookie()：清除Cookie

2.5 Router

router.all()：最好使用app.all()来代替router.all()。  
router.METHOD()：提供了路由功能。  
其中METHOD是小写的HTTP方法之一，如GET，PUT，POST等。  
因此，实际的方法是router.get()，router.post()，router.put()等。  
router.param()：设置路由参数。  
router.route()：返回单个路由的实例。  
router.use()：主要是针对指定路径下中间件的处理。
