2. Express

先学会用，再学会意义~
express的几个核心的类（对象）：

2.1 express（前三个不常用）

express.json()  
express.static()   
express.urlencoded()   
express.Router()：创建一个新的路由器对象。

2.2 Application

例如

const express = require('express');   
const app = express();

方法：

2.3 Request

app.get('/', function (req, res) {
   res.send('Hello World');
})中的req 

req.query：获取URL的查询参数串   
req.route：获取当前匹配的路由    
req.protocol：获取协议类型   
req.path：获取请求路径

2.4 Response

app.get('/', function (req, res) {
   res.send('Hello World');
})中的res

res.send()：传送HTTP响应  
res.clearCookie()：清除Cookie

2.5 Router

router.all()：最好使用app.all()来代替router.all()。  
router.METHOD()：提供了路由功能。  
其中METHOD是小写的HTTP方法之一，如GET，PUT，POST等。  
因此，实际的方法是router.get()，router.post()，router.put()等。  
router.param()：设置路由参数。  
router.route()：返回单个路由的实例。  
router.use()：主要是针对指定路径下中间件的处理。
